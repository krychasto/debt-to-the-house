# Debt to the House

Debt to the House is a Godot 4 roguelike blackjack deckbuilder. The prototype currently focuses on clean gameplay foundations before graphics.

## Current milestone

Sprint 1 adds the blackjack foundation: card data, a standard deck, hand scoring with flexible ace rules, configurable blackjack rules, result constants, a UI-independent blackjack engine, and basic run-level state for money, debt, hands, stages, tokens, and payouts.

Sprint 1 now also includes a playable Balatro-like 2.5D table scene with first-pass table and card art, energetic visual card panels, a hidden dealer hole card, bet input, Deal, Hit, and Stand controls, basic retry flow, and first rule-modifying relic rewards between stages. An experimental static-camera 3D scene is kept as reference.
